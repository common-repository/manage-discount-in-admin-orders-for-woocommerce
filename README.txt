This plugin allows you to manage discounts in WooCommerce orders placed in the backoffice.

== Description ==

With this plugin you'll be able to manage discounts in WooCommerce orders placed in the backoffice. It is possible to add discounts in line, per each product, or apply the discount for the total order.

== Installation ==
